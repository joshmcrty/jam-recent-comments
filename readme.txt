== Description ==

JAM Recent Comments creates a simple recent comments widget that displays the commenter's gravatar and a comment excerpt. You can choose the number of comments displayed.

== Installation ==

Upload the JAM CSS Naked Day plugin to your blog, Activate it, then set up your options (or leave the defaults).
